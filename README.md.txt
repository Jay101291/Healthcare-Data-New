# Insurance Premium Prediction with Streamlit

This project is a Streamlit web app for predicting insurance premiums using a trained XGBoost model.

## Features
- Upload your dataset for preprocessing.
- Predict annual premium amounts based on user inputs.
- Visualize feature importance.

## How to Run Locally
1. Clone this repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
